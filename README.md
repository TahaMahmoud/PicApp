# PicApp

PicApp is a simple iOS application, it retrieve pictures from free API and display them for  the user, and allow the user to search and display specific pictures.

Using [Pexels API](https://www.pexels.com/api/documentation/)

## Used In The APP
- SWIFT
- Alamfoire-based Network Layer
- MVVM Architecture Pattern with Closures
- Adobe XD: UI/UX Design for the Application
